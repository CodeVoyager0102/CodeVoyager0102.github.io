新一代效率工具平台---uTools

发表于2024-05-08
|字数总计:486|阅读时长:1分钟|阅读量:|评论数:

介绍

uTools是一款非常流行的桌面工具集合软件，它提供了丰富的功能和插件.用户可以根据自己的需求安装各种插件来扩展其功能，使得在日常工作中更加高效地进行各种操作。

安装步骤

  • 进入官网：https://www.u.tools/ 下载客户端

  • 选择符合自己电脑的来下载

  • 然后双击点我同意，注意这个改不了路径，直接就下载到了C盘。。。

相关使用

  • alt + 空格键: 唤醒搜索框(如果快捷键和其他应用冲突，可以去uTools里面修改一下快捷键：设置=>搜索框=>呼出快捷键)

  • 按鼠标滚轮，会弹出超级面板

  • 选中文本，按鼠标滚轮，会出现响应的插件可选用

  • 直接在主搜索框搜索相关内容打开十分方便

  • … …

相关插件

无需单独安装，直接 在uTools中安装需要的插件即可

在插件应用市场下载相关的插件，常用推荐：

Markdown笔记

右上角小飞机图标可以设置uTools关键字，外部可以快速打开笔记

json编辑器

JSON格式化处理

json格式化

可以格式化转化，生成接口等

粘贴板

可以查看历史粘贴的信息

备忘快贴

开启‘允许推送内容到搜索面板’，在注搜索框就能直接搜索到备忘信息

知犀思维导图

免费制图，导出文件及图片均无水印

聚合翻译

以浏览器方式聚合在线翻译，有三种：

本地搜索

这个搜索真的超级快速，平常在电脑本地搜索一个文件花费好长时间，这个就真的超级赞

git helper

可以快速查看git命令

还有其他很多功能强大的插件，感兴趣可以再去了解~
